### 1. Double Fetch — `web/src/app/[locale]/(main)/movies/[slug]/page.tsx:~20,60`
**Rule**: `async-parallel` (react-best-practices) + `server-cache-react` (react-best-practices)
**Problem**: `generateMetadata` và `Page` component đều gọi `movieService.getMovieBySlug(slug)` độc lập — request gọi 2 lần cho cùng 1 dữ liệu.
```tsx
// before — hai lần fetch cùng slug
export async function generateMetadata({ params }) {
  const movie = await movieService.getMovieBySlug(params.slug); // fetch #1
  ...
}
export default async function Page({ params }) {
  const movie = await movieService.getMovieBySlug(params.slug); // fetch #2 — duplicate!
  ...
}

// after — dùng React.cache() để deduplicate
import { cache } from 'react';
const getMovie = cache((slug: string) => movieService.getMovieBySlug(slug));

export async function generateMetadata({ params }) {
  const movie = await getMovie(params.slug); // cache hit on 2nd call
  ...
}
export default async function Page({ params }) {
  const movie = await getMovie(params.slug); // same request, deduped
  ...
}
```

### 4. Tất Cả Slides Render Đồng Thời — `web/src/components/movie/HeroCarousel.tsx:~30`
**Rule**: I2 (web-design), `bundle-dynamic-imports` (react-best-practices)
**Problem**: Loop render tất cả slides với `opacity-0` — tất cả backdrop images đều được tải về dù chỉ slide đầu hiển thị. Với 10+ phim trending, đây là ~10 large images tải không cần thiết.
```tsx
// before — tất cả slides trong DOM
{movies.map((movie, index) => (
  <div key={movie.id} className={`absolute inset-0 transition-opacity ${index === currentIndex ? 'opacity-100' : 'opacity-0'}`}>
    <Image src={movie.backdropUrl} ... />
  </div>
))}

// after — chỉ render slide hiện tại và adjacent slides
{movies.map((movie, index) => {
  const isVisible = Math.abs(index - currentIndex) <= 1;
  if (!isVisible) return null; // hoặc render placeholder
  return (
    <div key={movie.id} className={`absolute inset-0 transition-opacity ${index === currentIndex ? 'opacity-100' : 'opacity-0'}`}>
      <Image src={movie.backdropUrl} priority={index === currentIndex} ... />
    </div>
  );
})}
```

### 5. RecommendationRow Infinite Skeleton Khi API Lỗi — `web/src/components/movie/RecommendationRow.tsx:~40`
**Rule**: `rendering-conditional-render` (react-best-practices), C3 (web-design)
**Problem**: Nếu API throw error, `setLoading(false)` không bao giờ được gọi — component bị kẹt ở skeleton state mãi mãi (broken UX).
```tsx
// before — không có error handling
const load = async () => {
  if (cancelled) return;
  setLoading(true);
  const data = await fetchFn();   // nếu throw → setLoading(false) bị skip
  if (!cancelled) {
    setMovies(data);
    setLoading(false);
  }
};

// after — try/catch + error state
const [error, setError] = useState(false);
const load = async () => {
  if (cancelled) return;
  setLoading(true);
  setError(false);
  try {
    const data = await fetchFn();
    if (!cancelled) setMovies(data);
  } catch {
    if (!cancelled) setError(true);
  } finally {
    if (!cancelled) setLoading(false);
  }
};

// Trong JSX — hiển thị empty state khi error
if (error || (!loading && movies.length === 0)) return null;
```
